# 🍔 LezzetLab

LezzetLab, kullanıcıların hızlı ve kolay şekilde yemek siparişi verebilmesi için hazırlanmış bir yemek sipariş sitesi projesidir.  
Bu proje HTML ve CSS kullanılarak geliştirilmiştir.

## 🚀 Proje Özellikleri

- Ana sayfa
- Menü sayfası
- Restoranlar sayfası
- Kampanyalar sayfası
- İletişim sayfası
- Sayfalar arası geçiş sistemi
- Responsive tasarıma yakın düzen
- Video içerik
- Duyurular bölümü
- Kullanıcı yorumları
- Yemek kartları ve sipariş butonları

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3

## 📂 Sayfalar

- `index.html`
- `menu.html`
- `restoranlar.html`
- `kampanyalar.html`
- `iletisim.html`

## 🎯 Proje Amacı

Bu proje okul performans ödevi kapsamında hazırlanmıştır.  
Amaç, çok sayfalı bir web sitesi oluşturarak HTML ve CSS kullanımını geliştirmektir.
